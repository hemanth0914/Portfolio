My Work

The Secure File Share Application is built on AWS to provide a secure, scalable platform for file storage and transfer. It utilizes an Amazon S3 bucket for durable, always-available storage; an EC2 instance for processing and S3 bucket interaction, secured with an IAM role; an API Gateway to manage user requests and secure data transfer; a VPC link for direct, private API communication with the EC2 instance; and AWS Cloud Map for efficient service discovery and load management. This setup ensures seamless file uploads and downloads, robust scalability, and enhanced security, making it ideal for handling high volumes of user requests.
